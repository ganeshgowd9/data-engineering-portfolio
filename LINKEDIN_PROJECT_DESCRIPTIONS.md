# LinkedIn Project Descriptions

## Azure + Snowflake Data Implementation Pipeline
Built an end-to-end data implementation pipeline to ingest, validate, transform, and publish customer, policy, and claims-style datasets for analytics and reporting. Includes Azure Data Factory orchestration, Python validation, Snowflake SQL models, source-to-target mapping, and lineage documentation.

**Skills:** Azure Data Factory, Azure Data Lake, Snowflake, SQL, Python, Data Quality, Metadata Management, Data Lineage, ETL/ELT

## PySpark Lakehouse Claims Pipeline
Developed a PySpark-based lakehouse pipeline using Bronze, Silver, and Gold layer concepts. Demonstrates raw ingestion, cleansing, validation, transformation, and reporting-level aggregation for insurance-style claims data.

**Skills:** PySpark, Spark SQL, Databricks, Delta Lake Concepts, Data Lakehouse, ETL/ELT, Data Quality, Spark Optimization

## Airflow Data Quality Orchestration
Created an Apache Airflow DAG to orchestrate a production-style workflow with extraction, validation, transformation, publishing, and notification steps.

**Skills:** Apache Airflow, Python, Workflow Orchestration, Data Quality, Monitoring, Scheduling, Pipeline Reliability
