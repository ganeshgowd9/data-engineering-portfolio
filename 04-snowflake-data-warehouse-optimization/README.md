# Snowflake Data Warehouse Optimization

## Overview

This project demonstrates a Snowflake data warehouse design for customer, policy, and claims-style datasets. It includes staging tables, dimensional models, fact tables, and optimized reporting queries.

## Business Use Case

Analytics teams need fast and reliable reporting on policy and claims performance. This project shows how raw data can be modeled into clean reporting structures using Snowflake SQL.

## Skills Demonstrated

- Snowflake SQL
- Staging, dimension, and fact table design
- Data warehouse modeling
- Reporting-layer development
- Query optimization concepts
- Data mart design
- Analytics-ready datasets
