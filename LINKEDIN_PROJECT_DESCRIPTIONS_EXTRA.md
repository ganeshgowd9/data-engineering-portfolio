# Extra LinkedIn / GitHub Project Descriptions

## Snowflake Data Warehouse Optimization

Designed a Snowflake-based data warehouse and reporting layer for customer, policy, and claims-style datasets. The project includes staging tables, dimensional modeling, fact tables, SQL transformations, and query optimization concepts for analytics-ready reporting.

**Skills:** Snowflake, SQL, Data Warehousing, Dimensional Modeling, Query Optimization, Reporting Layer, Data Marts

## Kafka Streaming Claims Pipeline

Built a streaming data pipeline simulation for real-time claims/event ingestion. The project includes a Python event producer, consumer logic, JSON event structure, and documentation showing how streaming data can flow into downstream data platforms for monitoring and analytics.

**Skills:** Kafka, Python, Streaming Data, Event Processing, JSON, Real-Time Data Pipelines, Data Engineering

## Data Quality Framework

Created a reusable Python-based data quality framework to validate incoming datasets before transformation and reporting. The framework checks schema, null values, duplicates, accepted values, record counts, and reconciliation rules.

**Skills:** Python, SQL, Data Quality, ETL Testing, Data Validation, Reconciliation, Data Governance

## DevOps CI/CD Data Pipeline

Implemented a DevOps-style CI/CD workflow for data engineering projects using GitHub Actions. The workflow validates Python syntax, checks SQL files, and supports production-style release readiness for data pipeline code.

**Skills:** GitHub Actions, CI/CD, DevOps, Git, Python, SQL, Data Pipeline Deployment, Code Quality
